# Tower 13: Collapse Protocol

A grid-based escape puzzle game where players navigate a collapsing tower using logic and deduction.

## Features (Planned)

- Multi-floor system
- Logical danger detection
- Clue-based gameplay
- Puzzle rooms
- Inventory system
- Dynamic difficulty

## Current Features

- Grid system
- Player movement
- Basic UI layout

## Tech Stack

- HTML
- CSS
- JavaScript

## Future Scope

- Danger system (Minesweeper-like hints)
- Zombie / trap rooms
- Canvas-based animations
- Sound design

## How to Run

Open `index.html` in your browser.
